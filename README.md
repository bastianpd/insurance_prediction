## 📊 Project Overview

This project leverages **Machine Learning** and **Explainable AI (XAI)** techniques to predict healthcare insurance charges. We built a **Random Forest Regressor** and an **MLP Neural Network** to compare performance, achieving:

- **Random Forest:** R² of 97.43% (train) and 86.43% (test)
- **MLP Neural Network:** R² of 85.21% (train) and 87.01% (test)

### 🔍 Key Features
- **Explainability:** SHAP and LIME for model interpretability
- **Model Comparison:** Random Forest vs. MLP
- **Interactive Dashboard:** Tableau Public for data exploration
